# 工程目录结构

## 目录结构

```
├─src						工程文件夹
│  ├─Client					客户端文件夹
│  │  ├─java					源代码
│  │  ├─res						资源
│  │  └─test					测试代码
│  ├─jar					依赖
│  ├─Public					共有文件夹
│  │  ├─java					源代码
│  │  ├─res						资源
│  │  └─test					测试代码
│  └─Server					服务端文件夹
│     ├─java					源代码
│     ├─res						资源
│     └─test					测试代码
└─target					生成文件夹/*不提交到git仓库*/
    ├─classes
    └─generated-sources
        └─annotations
```
